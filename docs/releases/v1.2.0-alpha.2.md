# Faye v1.2.0-alpha.2

Release date: 2026-02-20

This prerelease supersedes `v1.2.0-alpha.1` with reliability corrections found during live launch execution.

## Fixes

1. macOS service status scripts now use robust `launchctl list <label>` checks.
2. Fixed false "stopped" reports caused by `pipefail` + `grep -q` behavior.
3. `doctor` now detects `rec` via fallback binary paths on macOS (`/opt/homebrew/bin`, `/usr/local/bin`, etc.).
4. Always-on proof script now safely handles workspace paths with spaces.

## Verification

1. Always-on proof report: services running + health `ok: true`.
2. Burn-in day rerun passed (`npm run canary`, Seven Shadow double pass, audit).

curl -fsSL https://raw.githubusercontent.com/VontaJamal/faye/main/scripts/bootstrap.sh | bash
