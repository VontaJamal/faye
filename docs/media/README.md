# Faye Media Pack

Use this folder for public onboarding visuals.

## Recommended Files

- `faye-onboarding.gif` (15-45 seconds, no audio)
- `faye-setup-walkthrough.md` (video link + short timestamps)

## Content Checklist

1. Show install command.
2. Show setup command.
3. Show dashboard opening.
4. Show one successful speech output.
5. Show service status check.

## Suggested Script

1. Terminal: `./scripts/install.sh`
2. Terminal: `./scripts/faye setup`
3. Browser: open `http://127.0.0.1:4587`
4. Terminal: `./scripts/speak.sh "Faye demo ready"`

Keep this short and clear for non-technical users.
